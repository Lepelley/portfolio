Nom du projet : Fly for Hell

  • Site web : ce site n'est plus disponible
  • Rôle : site d'un jeu vidéo où les membres pouvaient s'inscire, gérer leur compte et où les administrateurs pouvaient gérer les comptes des joueurs
  • Date de parution : ~2010
  • Langages utilisés : XHTML 1.0 Strict, CSS 2.1, PHP, Microsoft SQL Server.
  • Fonctionnalités : Mini-Blog, page d'inscription, classement des joueurs, système de gestion de compte, système d'administration (liste des membres, gestion des droits des membres, etc.)
